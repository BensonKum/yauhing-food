# 刪除產品流程

> **命名**：刪除產品 = 從網店/倉庫系統移除產品，包括前台卡片、JSON 資料、庫存記錄
> **觸發關鍵字**：「刪除產品」「下架產品」「移除產品」「停售產品」

## 適用場景
- 產品停產 / 唔再賣
- 暫停售售賣（隱藏但保留記錄）
- 清理重複 / 錯誤產品

---

## ⚠️ 重要決策：隱藏 vs 刪除

| 方式 | 效果 | 適用場景 |
|------|------|----------|
| **隱藏（推薦）** | 倉庫系統唔顯示，前台繼續賣 | 暫時缺貨、季節性產品 |
| **完全刪除** | 所有系統移除，歷史訂單可能受影響 | 停產、錯誤 SKU、重複項目 |

**預設建議用「隱藏」**，同鮑魚罐頭（YH401/YH401A）做法一致。

---

## 方案 A：隱藏產品（推薦）

### Step 1: products_v2.json 加 `"hidden": true`

```json
{
  "sku": "YH801",
  "name": "白菜鮮肉餃 (8隻裝)",
  "cat": "餃子系列",
  "price": "HKD 25",
  "stock": 50,
  "image": "product_cabbage_dumpling.jpg",
  "local_img": "product_cabbage_dumpling.jpg",
  "hidden": true   // 加呢行
},
{
  "sku": "YH801A",
  "name": "白菜鮮肉餃+麵",
  "cat": "餃子系列",
  "price": "HKD 20",
  "stock": 30,
  "image": "product_dumpling_noodle.jpg",
  "local_img": "product_dumpling_noodle.jpg",
  "hidden": true   // 加呢行
}
```

### Step 2: inventory.html 過濾邏輯

`renderGrid()` 函數已內建過濾：

```javascript
// 只顯示冇 hidden 或 hidden === false 嘅產品
const visibleProducts = data.filter(p => !p.hidden);
```

**確認 inventory.html 有呢段過濾邏輯，如果冇就要加。**

### Step 3: index.html（前台網店）

**唔使改**，前台繼續顯示同售賣。

如果前台都要隱藏，加 `data-hidden="true"` 屬性：

```html
<div class="p-card" data-sku="YH801" data-cat="餃子系列" data-hidden="true">
```

同埋喺 JS 過濾：

```javascript
document.querySelectorAll('.p-card[data-hidden="true"]').forEach(el => el.remove());
```

### Step 2: 刪除 index.html 卡片

```html

```

### Step 3: 刪除 inventory.html（自動）

**如果 products_v2.json 冇咗個 SKU，inventory.html 會自動唔顯示**（因為 `renderGrid()` 係從 JSON 讀取）。

### Step 4: 刪除 Firestore 庫存記錄

```javascript
// Firebase Console 或 Admin SDK
db.collection('inventory').doc('白菜鮮肉餃 (8隻裝)').delete()
```

**注意：Firestore doc ID 係產品名稱（中文），唔係 SKU。**

### Step 5: 清理圖片（可選）

```bash
# 如果圖片唔再用
git rm images/product_cabbage_dumpling.jpg
```

## AB 卡刪除特別注意

如果係 AB 卡（兩個 SKU 合併一張卡）：

| 刪除方式 | 影響 |
|----------|------|
| 兩個都刪 | 成張卡消失 |
| 淨刪 base SKU | Pack A 變孤兒，獨立顯示 |
| 淨刪 Pack A | Base SKU 變單卡 |

**建議：兩個 SKU 同時刪除或同時隱藏。**

## 完整檢查清清單

刪除產品時，依序檢查：

### 隱藏方案
- [ ] **products_v2.json**：加 `"hidden": true`
- [ ] **inventory.html**：確認有過濾邏輯
- [ ] **index.html**：如需前台隱藏，加 `data-hidden="true"`
- [ ] **Git**：commit + push
- [ ] **Firebase**：deploy hosting
- [ ] **測試**：倉庫系統唔顯示、前台（如設定）唔顯示

### 完全刪除方案
- [ ] **備份**：products_v2.json 另存備份
- [ ] **products_v2.json**：刪除條目
- [ ] **index.html**：刪除卡片
- [ ] **Firestore**：刪除庫存 doc（doc ID = 產品名稱）
- [ ] **圖片**：如唔再用，`git rm`
- [ ] **BOGO 列表**：如適用，從 `*_NAMES_1PLUS1` 陣列移除
- [ ] **Git**：commit + push
- [ ] **Firebase**：deploy hosting
- [ ] **測試**：前台、倉庫系統都唔顯示
