# 祐興食品 — 員工取貨核實操作指南

---

## 一、客人到店取貨流程

### 步驟 1：客人出示證明
客人會提供以下其中一種證明：
1. **WhatsApp 訊息** — 包含訂單編號（格式：YH + 時間戳，例如 `YH20260413234567`）
2. **QR Code 截圖** — 客人付款後網站生成的 QR Code
3. **訂單編號** — 客人直接口述訂單編號

### 步驟 2：打開核實工具
在手機或電腦瀏覽器打開：
```
https://bensonkum.github.io/yauhing-food/verify.html
```

### 步驟 3：輸入訂單編號
在搜尋框輸入訂單編號（例如 `YH20260413234567`），按「查詢」

### 步驟 4：核對訂單資料
系統會顯示以下資訊：

| 欄位 | 說明 |
|------|------|
| 訂單編號 | 格式：YH + 時間戳 |
| 客戶名稱 | 核對客人身份 |
| 聯絡電話 | 可要求客人出示手機確認 |
| 商品明細 | 確認貨品款式、數量 |
| 訂單金額 | 確認付款金額 |
| 付款狀態 | ✅ 已付款 / ⚠️ 待付款 |

### 步驟 5：確認付款狀態

| 狀態 | 顯示 | 操作 |
|------|------|------|
| **已付款** | ✅ 綠色橫幅 | 可以交貨 |
| **待付款** | ⚠️ 黃色警示 | 請客人先付款 |

### 步驟 6：交貨
- 核對無誤後，將貨品交給客人
- 客人簽收（如有需要）

## 二、QR Code 核實方式

### 客人 QR Code 包含什麼？
```
YH-PICKUP:{訂單編號}|{客戶名稱}|{電話}|HK${金額}
```
例如：`YH-PICKUP:YH20260413234567|陳小明|91234567|HK$118`

### 如何使用 QR Code？
1. 客人出示 QR Code 截圖
2. 目視核對 QR Code 上的資料與訂單系統顯示一致
3. 或直接輸入訂單編號查詢

---

## 三、常見情況處理

### 情況 1：客人找不到 WhatsApp 訊息 / QR Code
→ 請客人提供**訂單編號**或**註冊電話**，在系統查詢

### 情況 2：顯示「待付款」
→ 請客人先完成付款（轉數快 / 微信支付 / 支付寶），再重新查詢

### 情況 3：訂單不存在
→ 確認訂單編號是否正確，或聯絡負責人

### 情況 4：商品缺貨
→ 記錄缺貨項目，聯絡負責人處理退款或補貨
