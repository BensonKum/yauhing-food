# yauhing-food HTML Grid Layout 修復

## Objective
修復網站產品 grid 完全空白、layout 破碎問題

## 問題分析
- Ben 截圖顯示「特色醸」類別 8 款產品但 grid 空白
- 根因：index.html products section 有 **2 個未閉合 `<div>` 標籤**
- `<div` 數量：321，`</div>` 數量：319，差額 = 2
- 未閉合的是 `pg`（產品 grid 容器）和 `pi`（產品區容器）
- 導致瀏覽器解析時 grid container 提前關閉，所有產品卡跌出 CSS grid layout

## 修復
- 位置：`</section>`（products section 結尾）前
- 操作：插入 `</div></div>` 閉合 pg + pi 容器
- 結果：321 = 321，div 平衡 ✅
